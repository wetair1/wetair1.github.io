Source of my personal landing page — a monochrome, bilingual (EN / RU) page built with pure HTML / CSS / JS. No frameworks, no build step.

🔗 **Live:** https://wetair1.github.io

## Features

- Pure HTML / CSS / JS — zero dependencies, no build step
- Monochrome (grayscale) terminal aesthetic
- **EN / RU** language toggle
- Animated typewriter terminal + reveal-on-scroll project cards
- Lists all my terminal projects with links

## Develop

Open `index.html` in any browser — that's it. Deployed automatically via GitHub Pages from the `main` branch.
